# Barème sur 20 points

| Critère | Attendus | Points |
|---|---|---:|
| 1. Mise en place de l'environnement | Le lab est lancé correctement, les scripts sont utilisés correctement, l'environnement de travail est fonctionnel. | /2 |
| 2. Génération des clés | Les paires de clés Alice et Bob sont créées correctement. | /2 |
| 3. Export et import des clés publiques | Les clés publiques sont bien exportées et importées dans les bons environnements. | /2 |
| 4. Signature d'un message | Le message signé est produit correctement. | /2 |
| 5. Vérification de signature | L'étudiant sait vérifier la signature et interpréter le résultat. | /2 |
| 6. Chiffrement du message | Le message est correctement chiffré pour Bob. | /2 |
| 7. Déchiffrement du message | Le message est correctement déchiffré par Bob. | /2 |
| 8. Signature + chiffrement | L'étudiant réalise correctement la combinaison signature + chiffrement. | /2 |
| 9. Analyse technique | L'étudiant distingue bien confidentialité, intégrité, authenticité, clé publique, clé privée, empreinte. | /3 |
| 10. Analyse juridique et professionnelle | L'étudiant identifie correctement les limites de PGP/GPG et son apport dans les moyens de preuve. | /3 |

# Aide à l'évaluation détaillée

## 1. Mise en place de l'environnement /2
- 0 : environnement non fonctionnel
- 1 : environnement partiellement fonctionnel
- 2 : environnement correctement utilisé

## 2. Génération des clés /2
- 0 : non réalisé
- 1 : partiellement réalisé
- 2 : réalisé correctement

## 3. Export / import des clés publiques /2
- 0 : non réalisé
- 1 : réalisé avec erreurs
- 2 : réalisé correctement

## 4. Signature d'un message /2
- 0 : non réalisé
- 1 : partiellement correct
- 2 : correct

## 5. Vérification de signature /2
- 0 : non réalisé
- 1 : résultat observé sans explication
- 2 : résultat compris et expliqué

## 6. Chiffrement du message /2
- 0 : non réalisé
- 1 : partiellement réalisé
- 2 : correct

## 7. Déchiffrement du message /2
- 0 : non réalisé
- 1 : partiellement réalisé
- 2 : correct

## 8. Signature + chiffrement /2
- 0 : non réalisé
- 1 : partiellement réalisé
- 2 : correct

## 9. Analyse technique /3
- 0 : analyse absente ou erronée
- 1 : analyse incomplète
- 2 : analyse correcte mais peu développée
- 3 : analyse claire, structurée et pertinente

## 10. Analyse juridique et professionnelle /3
- 0 : aucune analyse
- 1 : analyse très incomplète
- 2 : analyse correcte
- 3 : analyse pertinente avec recul professionnel
